## SegFormer B0 vs B2 성능 비교 (CamVid 384×288, Fold1↔Fold2 Cross Test)

### 1. 실험 설정

- **Dataset**: CamVid 384×288, FOLD1/FOLD2 교차 평가  
  - Train: 각 fold의 `train` 만 사용  
  - Val/Test: 반대 fold의 `train + val` 전체 사용
- **모델**
  - **B0 (Student)**: `mit_b0` backbone, ImageNet pretrained
  - **B2 (Teacher)**: `mit_b2` backbone, ImageNet pretrained
- **평가 방식**
  - Fold1 모델 → Fold2 전체 평가
  - Fold2 모델 → Fold1 전체 평가
  - 두 방향 결과를 평균내어 최종 성능 산출
